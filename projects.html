SAMPLES

Here I will be demonstrating some functions that I have implemented in my projects. These examples will show my use of DOM programming and utilizing Application Programming Interface to fetch and manipulate data.

Bitcoin Value

In this function I Access a web service at coindesk.com to obtain an API that allows me to fetch the current value of Bitcion. This function will automatically update the value every minute, you can also click the button to make sure its up to date.

Joke Generater

This function Generates Chuck Norris Jokes using an API from chucknorris.io. In this function you can either get a random joke, or get a random joke from a specific category.

Generate images

When this page gets opened the following function fetches a list of available dog breeds and updates the function if new breeds were added or removed from the API. If the user picks a breed and clicks the submit button the function will display photos of that breed.
